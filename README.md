# Podtrženo.cz | Chytré finanční kalkulačky

**Živá verze aplikace:** https://podtrzeno.cz/

---

### O projektu
Jednoduchá a přehledná frontendová aplikace zaměřená na finanční gramotnost. Obsahuje interaktivní kalkulačky pro výpočet splátek, úroků a přehled aktuálních kurzů.

### Technologický stack
* **Frontend:** HTML, JavaScript
* **Styling:** SCSS (vlastní design systému)
* **Výkon:** Optimalizováno pro Lighthouse (skóre 100)
